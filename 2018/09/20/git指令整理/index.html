git指令整理

基本操作

  • git init 创建仓库
  • git clone xxxx 拉取项目
  • git branch -a 查看本地分支
  • git checkout -d xxx 创建本地分支并切换到该分支
  • git add -A 追踪本地修改过的全部文件
  • git commit -m “xxx” 提交本地更新
  • git pull origin xxx 拉取远程分支
  • git push origin xxx 推送本地更新

远程关联或取消

  • 关联远程仓库 ：
    • git init
    • git remote add origin xxx
    • 注意这里的xxx其实就是指的这个 ：
  • 取消本地关注 ：
    • git remote remove origin

回滚相关

  • git add 后回滚到上一次（未提交） ：
    • git reset HEAD xxxx 撤销本次更新
    • git checkout – xxxx 回滚到上一次提交
  • git commit 后回滚到上一次提交 ：
    • git log 查看全部提交的日志信息
    • git reset –hard xxxx 回滚到xxxx(commit号)那次提交

分支合并

  • 将新分支上的代码合并到master ：
    • 切换到master分支后，
    • git merge xxx
    • 最后可以 git branch -d xxx 删掉这个新分支

远程分支

  • 拉取远程分支上的内容 ：
    • git checkout -b xxx 新建并切换本地分支
    • git pull origin xxx 拉取远程该分支上的内容
  • 在本地新建分支并推送到远程 ：
    • git checkout -b xxx 新建并切换到本地分支
    • git push origin xxx 直接将该分支推送过去
  • 删除某一分支 ：
    • 首先切换到其他分支，然后，
    • 本地 ：git branch -d xxx
    • 远程 ：git push origin –delete xxx
